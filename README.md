## Install
```bash
$ npm install zmethods --save
```

## Build
```bash
$ npm run build
```

## Test
```bash
$ npm run test
```

## Methods

|||
|----|----|
|debounce|防抖|
|throttle|节流|
|sleep|睡眠|
|arrUnique|数组去重|
|toIndexOf|判断是否在数组中|
|byteFormat|byte换算|
|thousandsDot|数组每三位逗号分隔|
|deepCopy|对象/数组深拷贝|
|guid|生成guid|
|numPadding|数字补零|
|findLackNum|查找顺序数字中缺失的数字|
|arrChunk|数组按指定大小分割|
|isJSON|判断是否是JSON|
|arrShuffle|打乱数组|
